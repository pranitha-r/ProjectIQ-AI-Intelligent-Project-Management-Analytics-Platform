# ProjectIQ — Intelligent Project Management & Analytics Platform

License: MIT • Node.js • Express • SQLite • JavaScript

An intelligent project management and analytics platform built with Node.js, Express, SQLite, and JavaScript. ProjectIQ helps teams manage projects, track tasks, monitor workload distribution, forecast completion timelines, and identify project risks through interactive dashboards and data-driven insights.

---

## ✨ Project Overview

ProjectIQ is designed to help project managers and teams improve project visibility, resource planning, and decision-making through centralized project tracking and analytics.

The platform combines project management, workload monitoring, completion forecasting, and risk analysis into a single dashboard-driven experience.

### Why this project exists

* Help teams organize projects and tasks efficiently
* Improve visibility into project progress and team performance
* Detect workload imbalances and resource bottlenecks
* Forecast project completion timelines
* Identify project risks before they impact delivery
* Provide actionable insights through analytics dashboards

## 🛠️ Install & Setup

### Prerequisites

* Node.js 18+
* npm

### Clone Repository

```bash
git clone https://github.com/yourusername/projectiq.git
cd projectiq
```

### Install Dependencies

```bash
npm install
```

### Run the Application

```bash
node server.js
```

Open:

```text
http://localhost:3000
```
